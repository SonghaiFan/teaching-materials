# Teaching Materials

Course materials repository organized by institution and course code.

## Structure

```
materials/
├── lincoin/              # External tutoring materials
│   └── fit5196/          # FIT5196 - Data Visualisation
│       ├── week1/
│       ├── week2/
│       └── ...
│
└── monash/               # Monash University materials
    ├── fit2179/          # FIT2179 - Data Visualisation
    │   ├── week1/
│   │   ├── week2/
│   │   └── ...
    │
    └── fit5196/          # FIT5196 - Data Visualisation
        ├── week1/
        ├── week2/
        └── ...
```

## Contents

Each course folder may contain:

- **week*/** - Weekly materials
  - Lecture notes
  - Tutorial exercises
  - Assignment briefs
  - Reading materials
  - Sample solutions
  - Code examples
  - Datasets

- **assessments/** - Assessment materials
- **readings/** - Required and recommended readings
- **datasets/** - Data files for exercises
- **code/** - Code templates and examples

## Usage

1. Navigate to the relevant course folder
2. Find materials organized by week or category
3. Add new materials as needed following the existing structure

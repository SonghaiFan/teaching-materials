## 快速操作

**新建课程：**
1. 在对应目录创建文件夹
2. 复制 `templates/syllabus_template.md`
3. 填写课程信息

**制作课件：**
1. 告诉我课程主题和目标
2. 我生成 slides 大纲
3. 你确认后，我填充详细内容
4. 导出到你喜欢工具（Google Slides / PowerPoint / Keynote）

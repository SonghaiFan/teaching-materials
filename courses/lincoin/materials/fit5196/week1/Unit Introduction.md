Unit Objectives
- FIT5196 is trying to achieve parse data in the required format; assess the quality of data for problem identification; resolve data quality issues ready for the data analysis process; integrate data sources for data enrichment; document the wrangling process for professional reporting; write program scripts for data wrangling processes.
- We DO NOT focus on
- Introduction to Python programming, e.g., how to program in Python
You NEED be very familiar with Python and the usage of Python packages!

## Page 10

Assessment 1 (35%)
### Exploratory Data Analysis (EDA)
- Data stored in various structures and formats need to be explored, pre-processed and analysed well to gain a comprehensive understanding.
- Summarise your findings from raw and pre-processed data and identify any potential interesting research problems/tasks in the further machine learning analysis.
- In this assessment, you may require to write Python (Python 3) script to
- Load and extract data from input file(s),
- Apply simple pre-processing operations on raw data, i.e. data format/type conversion,
- Apply EDA with appropriate visualizations
- One submission by a group of 4-5 people
- Due time: 23:55, Thursday Week 7
- No 2-day short extension is appliable for this group assessments, only Special Consideration

## Page 11

Assessment 2 (40%)
Parsing, Cleansing and Integrating Raw Data
- This assessment addresses the most important but time-consuming steps in data wrangling, i.e., cleansing and integrate data. Students are required to inspect, audit and then identify problems existing in the parsed data, and propose appropriate methods to fix these problems different generic and major data problems could be found in the data might include Lexical errors,
Irregularities, Violations of the Integrity constraint, Duplications, Inconsistency and more.

- Handle missing values
- Handle outliers
- One submission by a group of 4-5 people
- Report due time: 23:55, Thursday Week 12
- All group members need to present their work in-person in Week 15 (2nd week of exam period)
- Hurdle Applied to presentation
- No 2-day short extension is appliable for this group assessments, only Special Consideration

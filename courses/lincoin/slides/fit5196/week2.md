# Key Tasks Overview

---

# Main Wrangling Tasks

| Task | Description | Tools |
|------|-------------|-------|
| **Data Discovery** | Find and access data sources | APIs, Web scraping |
| **Structuring** | Organize unstructured data | Pandas, JSON |
| **Cleaning** | Handle errors and inconsistencies | Regex, Pandas |
| **Transformation** | Convert formats and types | NumPy, Pandas |
| **Enrichment** | Add value with external data | APIs, Joins |
| **Validation** | Verify data quality | Assertions, Tests |

# Tableau Key Concept

## Dual Axis

- Combine two measures into one view
- Example:
  - Bar → Average
  - Line → Maximum

## Why?

- Reduce visual clutter
- Increase data–ink ratio

---

# Common Problem

## Misleading Axis

- Dual axis can distort interpretation
- Example issue:
  - Max appears below Avg (wrong perception)

## Fix

- **Synchronise Axis**
- Ensure shared scale

---

# Design Principle

## Least Ink Principle

- Minimise visual noise
- Maximise information

### Example

- Replace colour with labels
- Use line-end labels instead of legend

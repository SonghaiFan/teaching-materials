My First Data Visualisation Web Page

This example shows a simple page structure using Bootstrap. It includes text, one image, one table, and one Vega-Lite chart.

A section with a title

In HTML, we often divide a page into sections. Each section can have its own heading and content.

A simple Vega-Lite bar chart

The chart below is embedded inside a normal HTML `div`.

An image in HTML

You can add an image with the `img` tag.

A list in HTML

  • Use `h1` and `h2` for titles
  • Use `p` for paragraph text
  • Use `ul` and `li` for bullet lists
  • Use `img` to show an image
  • Use a `div` to hold a chart

Why split HTML, CSS, and JS?

  • `index.html` holds the page structure
  • `style.css` holds the styling
  • `main.js` holds the JavaScript
  • `barchart.js` holds the Vega-Lite chart code

How this page is structured

  • `container` keeps the page centered
  • `row` creates a horizontal layout
  • `col-lg-8` and `col-lg-4` create two columns
  • `section` helps group content with a title
  • text is written with normal HTML elements like `h1`, `h2`, and `p`
  • `table`, `ul`, and `img` are basic HTML elements you will use often
  • the chart is added with JavaScript into `#vis`
